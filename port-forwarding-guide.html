🏠 方案2：家庭路由器端口转发

配置步骤

1 获取本地IP地址 
ifconfig | grep "inet "

找到您的本地IP地址（通常是 192.168.x.x 格式）

2 登录路由器管理界面

在浏览器中访问路由器IP（通常是 192.168.1.1 或 192.168.0.1）

3 找到端口转发设置

进入"高级设置" → "端口转发"或"NAT设置"

4 添加端口转发规则
  • 内部IP： 您的本地IP地址（如 192.168.1.100）
  • 内部端口： 8000
  • 外部端口： 8000（或选择其他端口）
  • 协议： TCP
5 保存设置

点击"保存"或"应用"按钮

6 获取公网IP

访问 ip138.com 获取您的公网IP地址

7 外网访问链接

使用 http://您的公网IP:8000 访问游戏

⚠️ 注意事项

  • 需要有公网IP地址（部分ISP可能使用NAT）
  • 可能需要配置防火墙允许端口访问
  • 公网IP可能会动态变化（可考虑使用DDNS）
  • 安全性需要考虑，建议设置强密码

🔄 方案3：使用内网穿透工具

使用frp进行内网穿透

1 准备一台云服务器

购买一台有公网IP的云服务器（阿里云、腾讯云等）

2 下载frp

在服务器和本地机器上都下载frp工具：

wget https://github.com/fatedier/frp/releases/download/v0.51.3/frp_0.51.3_linux_amd64.tar.gz
tar -xzf frp_0.51.3_linux_amd64.tar.gz
cd frp_0.51.3_linux_amd64
3 配置服务器端（frps）

编辑 frps.ini：

[common]
bind_port = 7000
vhost_http_port = 8080

启动服务器：

./frps -c frps.ini
4 配置客户端（frpc）

编辑 frpc.ini：

[common]
server_addr = 服务器公网IP
server_port = 7000

[web]
type = http
local_ip = 127.0.0.1
local_port = 8000
custom_domains = 您的域名或服务器IP

启动客户端：

./frpc -c frpc.ini
5 外网访问

使用 http://服务器IP:8080 访问游戏
