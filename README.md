# 儿童赛车游戏移动应用

一个适合5-12岁儿童的赛车游戏移动应用，支持iOS和Android平台。

## 功能特点

- 🎮 简单易用的触摸控制
- 🚗 多种可爱的赛车角色选择
- 🏆 计分和奖励系统
- 🌤️ 动态天气效果
- 💎 道具收集功能
- 📱 支持iOS和Android设备
- 🌐 离线游戏模式

## 安装说明

### Android 设备

1. **直接安装（推荐）**
   - 下载 `kids_racing_game.apk` 文件到您的Android设备
   - 打开文件管理器，找到下载的APK文件
   - 点击安装（可能需要允许"安装来自未知来源的应用"）
   - 安装完成后，点击桌面图标启动游戏

2. **通过构建脚本安装**
   ```bash
   # 确保已安装 Node.js、npm、Java JDK 和 Android SDK
   ./build-android.sh
   # 按照脚本提示完成签名和安装
   ```

### iOS 设备

1. **通过 TestFlight 安装**（需要开发者账号）
   - 联系开发者获取 TestFlight 邀请链接
   - 点击链接，按照提示安装 TestFlight 应用
   - 在 TestFlight 中安装儿童赛车游戏

2. **通过 Xcode 安装**
   ```bash
   # 确保在 macOS 系统上，已安装 Xcode 和 CocoaPods
   ./build-ios.sh
   # 按照脚本提示在 Xcode 中打开项目并运行
   ```

## 使用说明

1. **启动游戏**：点击手机桌面上的"儿童赛车"图标
2. **选择赛车**：在车库中选择您喜欢的赛车
3. **开始游戏**：点击"开始比赛"按钮
4. **控制方式**：
   - 触摸屏幕左右滑动控制赛车方向
   - 或使用屏幕底部的虚拟方向按钮
   - 部分设备支持倾斜控制
5. **游戏目标**：躲避障碍物，收集道具，获得高分

## 设备要求

- **Android**：Android 5.0 (API 21) 或更高版本
- **iOS**：iOS 12.0 或更高版本
- **存储空间**：至少 50MB 可用空间
- **内存**：至少 2GB RAM

## 常见问题

### 游戏无法启动
- 检查设备是否满足最低系统要求
- 尝试重启设备后再次启动游戏
- 确保已授予游戏所需的权限

### 控制不灵敏
- 进入设置界面校准控制灵敏度
- 确保屏幕清洁，无灰尘或水渍
- 尝试使用虚拟按钮控制

### 游戏卡顿
- 关闭其他正在运行的应用
- 降低游戏画质设置
- 确保设备电量充足

### 无法安装应用
- Android：检查是否允许"安装来自未知来源的应用"
- iOS：确保 Apple ID 已登录，且设备已授权
